⇑ Centrifugadoras

URI https://purl.org/pe-repo/concytec/equipamiento#centrifugadoras
Definition (es) Una centrifugadora es una máquina que pone en rotación una muestra para –por fuerza centrífuga– acelerar la decantación o la sedimentación de sus componentes o fases (generalmente una sólida y una líquida), según su densidad. Existen diversos tipos, comúnmente para objetivos específicos. Incluye centrífugas analíticas, de mesa, de alta capacidad, de alta velocidad, ultracentrifugadoras y microcentrigugadoras.
Preferred label (es) Centrifugadoras

⇑ Computadoras de alto desempeño

URI https://purl.org/pe-repo/concytec/equipamiento#computadorasDeAltoDesempeno
Definition (es) Sistemas de cómputo avanzado (súper computadoras) y de sistemas formados por varios procesadores interconectados. Estos equipos son capaces de procesar enormes cantidades de información, lo que permite hacer cálculos y simulaciones imposibles en equipos de cómputo convencionales. Ejemplo: Sistema Computacional de Alto Rendimiento para la Simulación de Fluidos. Fuente: Dirección de Investigaciones y Estudios del Concytec.
Preferred label (es) Computadoras de alto desempeño

⇑ Cromatógrafos

URI https://purl.org/pe-repo/concytec/equipamiento#cromatografos
Definition (es) Equipamientos científicos basados en el principio de retención selectiva, cuyo objetivo es separar los distintos componentes de una mezcla, permitiendo identificar y determinar las cantidades de dichos componentes. Incluye Cromatógrafos de Gases, Cromatógrafos Líquidos, entre otros. Fuente: Dirección de Investigaciones y Estudios del Concytec.
Preferred label (es) Cromatógrafos

⇑ Drones

URI https://purl.org/pe-repo/concytec/equipamiento#drones
Definition (es) Un vehículo aéreo no tripulado (VANT), comúnmente conocido como dron,​ hace referencia a una aeronave que vuela sin tripulación, la cual ejerce su función remotamente. Un VANT es un vehículo sin tripulación, reutilizable, capaz de mantener de manera autónoma un nivel de vuelo controlado y sostenido, y propulsado por un motor de explosión, eléctrico o de reacción.
Preferred label (es) Drones

⇑ Equipos de GPS

URI https://purl.org/pe-repo/concytec/equipamiento#equiposDeGPS
Definition (es) Equipos que permiten determinar en toda la Tierra la posición de cualquier objeto (una persona, un vehículo) con una gran precisión. Tienen muchas aplicaciones, entre ellas podemos mencionar, la navegación terrestre, marítima y aérea, la topografía y geodesia, la construcción, fines militares, entre otras.
Preferred label (es) Equipos de GPS

⇑ Equipos de imágenes médicas

URI https://purl.org/pe-repo/concytec/equipamiento#equiposDeImagenesMedicas
Definition (es) Equipos para crear imágenes del cuerpo humano, o partes de él, con propósitos clínicos (procedimientos médicos que buscan revelar, diagnosticar o examinar enfermedades) o para la ciencia médica (incluyendo el estudio de la anatomía normal y función). Incluye: equipos de ultrasonido, ecógrafos, resonancia magnética, tomógrafos, entre otros.
Preferred label (es) Equipos de imágenes médicas

⇑ Equipos de Procesamiento y Ensayo de Materiales

URI https://purl.org/pe-repo/concytec/equipamiento#equiposDeProcesamientoyEnsayoDeMateriales
Definition (es) Equipamiento para procesar materias primas y fabricación de productos (a escala de laboratorio, piloto o industrial), además de equipos para realizar ensayos de materiales. Fuente: Dirección de Investigaciones y Estudios del Concytec.
Preferred label (es) Equipos de Procesamiento y Ensayo de Materiales

⇑ Equipos de Rayos X

URI https://purl.org/pe-repo/concytec/equipamiento#equiposDeRayosX
Definition (es) Instrumentos que utilizan como fuente de emisión los Rayos X. Incluye: Difractómetros de Rayos X, Fluorescencia de Rayos X, entre otros. Fuente: Dirección de Investigaciones y Estudios del Concytec.
Preferred label (es) Equipos de Rayos X

⇑ Equipos sísmicos

URI https://purl.org/pe-repo/concytec/equipamiento#equiposSismicos
Definition (es) Dispositivos para la investigación geofísica, estudio sísmico, para el monitoreo automatizado de elementos estructurales de edificios y estructuras, monitoreo sísmico de oleoductos y gasoductos. Incluye sismógrafos, registradores, acelerómetros, equipos de refracción sísmica, equipos sísmicos digitales, entre otros.
Preferred label (es) Equipos sísmicos

⇑ Espectrómetros

URI https://purl.org/pe-repo/concytec/equipamiento#espectrometros
Definition (es) Equipamientos científicos que son utilizados para identificar materiales, analizando el tipo de espectro que emite una fuente o que es absorbida por una sustancia que se encuentra en el camino de la luz que emite una fuente. Incluye: Espectrómetros de Masa, Espectrómetros de Resonancia Magnética Nuclear, Espectrómetros de Fluorescencia, entre otros. Fuente: Dirección de Investigaciones y Estudios del Concytec.
Preferred label (es) Espectrómetros

⇑ Instrumentos Bioanalíticos

URI https://purl.org/pe-repo/concytec/equipamiento#instrumentosBioanaliticos
Definition (es) Corresponde a equipos de control, desarrollo, evaluación y análisis aplicados a muestras provenientes del área biológica. El equipamiento considerado en esta categoría es de carácter diverso: Secuenciadores de ADN, Autoanalizador de Bioquímica Clínica, Magnetoencefalógrafo, PCR, Cabina de bioseguridad, equipos especializados de inmunología tal como citómetros de flujo, entre otros. Fuente: Dirección de Investigaciones y Estudios del Concytec.
Preferred label (es) Instrumentos Bioanalíticos

⇑ Microscopios

URI https://purl.org/pe-repo/concytec/equipamiento#microscopios
Definition (es) Equipamientos científicos destinados a aumentar la imagen de los objetos. Incluye diferentes tipos de Microscopios tales como: Microscopio Confocal, Microscopio de Fluorescencia, Microscopio Electrónico Barrido, Microscopio de Fuerzas Atómicas, entre otros. Fuente: Dirección de Investigaciones y Estudios del Concytec.
Preferred label (es) Microscopios

⇑ Radares

URI https://purl.org/pe-repo/concytec/equipamiento#radares
Definition (es) El radar (término derivado del acrónimo inglés radio detection and ranging, “detección de radio y rango ”) es un sistema que usa ondas electromagnéticas para medir distancias, altitudes, direcciones y velocidades de objetos estáticos o móviles como aeronaves, barcos, vehículos motorizados, formaciones meteorológicas y el propio terreno
Preferred label (es) Radares

⇑ Refrigeradores

URI https://purl.org/pe-repo/concytec/equipamiento#refrigeradores
Definition (es) Equipos cuya función principal consiste en mantener, en un ambiente controlado (espacio refrigerado), diversos fluidos y sustancias, para que los mismos se conserven en buenas condiciones, mientras más baja sea la temperatura, menor actividad química y biológica. Incluye: refrigeradores de conservación, refrigeradores de baja temperatura, refrigeradores de ultrabaja temperatura, frigoríficos, congeladores, entre otros.
Preferred label (es) Refrigeradores

⇑ Transceptores

URI https://purl.org/pe-repo/concytec/equipamiento#transceptores
Definition (es) Dispositivo que cuenta con un transmisor y un receptor que comparten parte de la circuitería o se encuentran dentro de la misma caja.​ Cuando el transmisor y el receptor no tienen en común partes del circuito electrónico se conoce como transmisor-receptor.
Preferred label (es) Transceptores

Preferred label (es) Otros equipamientos
